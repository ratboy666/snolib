AGT - Alphabetic comparision with upper and lower equal

AGT(S1,S2) is the same as LGT(S1,S2) but considers upper and lower case identical. AGT converts both strings to lowercase. This is done by creating a translation table based on &ALPHABET.

After the strings have been translated, LGT is used for the comparision.

-PUBLIC AGT()
*
         DEFINE('AGT(S1,S2)')
         AGT_TT = REPLACE(&ALPHABET, &UCASE, &LCASE)          :(AGT_END)
*
AGT      LGT(REPLACE(S1, &ALPHABET, AGT_TT),
+            REPLACE(S2, &ALPHABET, AGT_TT))        :S(RETURN)F(FRETURN)
*
AGT_END
