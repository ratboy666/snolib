AI - Apply Index

AI(A,I) where A and I are arrays will regard I as a set of indices to be applied to the array A. The result is a new array of the selected elements. If I is a scalar, the result is A<I>.

AI() creates a new array. Each element of the new array is populated by the I<N> element of A. AI() uses SEQ() to step through A and I.

If I is not an array, simply returns A<I>.
