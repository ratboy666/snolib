FIND - Find extreme

FIND(A,PRED) will search array A for an extreme element. The type of extreme element will be determined by the predicate PRED. 


|     Examples: FIND(A,'GE')
|               FIND(A,'~LGT')

FIND     EX = CONVERT(PRED '(MAX,TEST)', 'EXPRESSION')
         FIND = 1
         MAX = A<FIND>
         I = 1
FIND_1   I = I + 1
         TEST = A<I>                                          :F(RETURN)
         EVAL(EX)                                             :S(FIND_1)
         MAX = TEST
         FIND = I                                              :(FIND_1)

FIND_END
