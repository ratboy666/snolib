JIT - Just In Time Code Generation

1580 Interface to the GNU Lightning library to allow dynamic code production at run-time for: ARM, HPPA, IA64, MIPS, PPC, S390, SPARC, Intel X86 (32 bit) and X86_64.

GNU Lightning does not have an ELF writer. It does appear to support all of the platforms that I would want.
