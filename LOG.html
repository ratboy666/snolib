LOG - LOG Functions

CLOG(X) will return the common log (base 10) of X.
LOGB(X,B) will return the logarithm of X to the base B.
LOGB(X) will return the natural logarithm of X (same as LOG(X)). Note that CSNOBOL4 also has LN(X) for natural log.
