LPROG - Length of program

Profiling and other tasks need to know the number of statements in the program. The LPROG() function supplies this information. Each statement compiled is given a new statement number. This is exploited by returning the statement number of a new statement.

LPROG() will return the length (i.e., the number of statements) in the SNOBOL4 program in which it is called. Every use of CODE() will increase this number, so call this early during execution.
