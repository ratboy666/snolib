MD5 - Return MD5 hash of string

MD5(S) returns the MD5 hash of the string S.

-PUBLIC MD5()
*
         DEFINE('MD5(S)T')                                    :(MD5_END)
*
MD5      T = MKTEMP()
         WRITEFILE(T, S)
         MD5 = BQ('md5sum ' T)
         MD5 BREAK(' ') . MD5
         DELETE(T)                                             :(RETURN)
*
MD5_END
@
