NOTP - Invert Pattern

NOTP(P) will return a pattern which will match the NULL string if P fails and fail if P matches. If P aborts, NOTP(P) will also abort.

Gimpel's version uses *PUSH() instead of PUSH(). I think that is wrong. If pattern P fails, backtracking will go further "left". That would cause extra NULLs to be pushed which would not be POPed. Using PUSH() instead helps. 


«NOTP»≡
NOTP     NOTP = PUSH() P *?(POP() PUSH(1)) FAIL |
+               *IDENT(POP())                                  :(RETURN)
*
NOTP_END
@
