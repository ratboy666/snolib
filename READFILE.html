READFILE - Read complete file into string

Reads a complete file into a string. No conversions are done. Uses 'B,65536' (binary, 64k byte buffer) for the INPUT() function. This is specific to CSNOBOL4.

This is used to bring in content from disk (say, images, templates or html). This content can then be left in memory if needed.

Tested w/ 1024..256K buffer size. Default 64K buffer.
