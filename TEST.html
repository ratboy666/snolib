TEST - Bypass one character assumption

TEST(ARG) accepts an unevaluated expression as argument, and returns a pattern bypasses SNOBOL4's one-character length assumption (quickscan heuristics).

It is never necessary to use TEST(ARG) with SPITBOL, because SPITBOL does not have a one-character assumption.
