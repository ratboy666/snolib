uban

Produces banners with line drawing characters in UTF-8. 13 of these fit into 80 print columns (although each column is 3 characters). Allowed characters are A..Z, 0..9, space and dash. Each character is formed on a 6X6 grid. Each of the characters used is actually a 3 byte UTF-8 code.

Uses

CRACK
SEQ
TRIMB
